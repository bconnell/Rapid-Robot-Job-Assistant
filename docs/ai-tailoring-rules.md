# AI Tailoring Rules

Tailoring must be truthful and based on user-provided data.

## Allowed

- Reorder existing skills.
- Suggest clearer wording for real experience.
- Highlight matching projects or tools already in the profile.
- Draft application answers from real profile/resume data.
- Flag missing keywords that are not supported by the profile.

## Not Allowed

- Invent degrees.
- Invent certifications.
- Invent employers.
- Invent dates.
- Invent years of experience.
- Claim tools the user does not know.
- Present suggestions as accepted facts.

## Review Flow

Suggestions start as suggestions. The user must accept, edit, or reject them. AI provider calls require a visible review step before sending data.
